📦 AlmaCen – Inventory Management System<br>

👩‍💻 Author<br>
Sharanya Aithal KS<br>

📌 Overview<br>

AlmaCen is a Web-Based Inventory Management System designed to simplify stock management and improve operational efficiency.<br>
The system provides a centralized platform where users can manage inventory items, track stock movements, and monitor real-time stock status effectively.<br>
Built using the MERN Stack, AlmaCen demonstrates a modern, scalable, and secure approach to inventory control, eliminating manual errors and improving decision-<br>making through automation and real-time updates.

🎯 Objectives<br>

-> Develop a web-based inventory management system<br>
-> Implement secure authentication and authorization (JWT)<br>
-> Provide centralized inventory control (Add, Edit, Delete items)<br>
-> Track stock inflow and outflow efficiently<br>
-> Display stock status (In Stock, Low Stock, Out of Stock)<br>
-> Implement full CRUD operations using MERN stack<br>
-> Ensure efficient database management using MongoDB<br>
-> Build a scalable and modular full-stack application<br>
-> Enhance user experience with responsive UI and real-time updates<br>

🛠️ Technologies Used<br>

-> JavaScript – Core programming language<br>
-> React.js – Frontend UI development<br>
-> Node.js – Backend runtime environment<br>
-> Express.js – REST API development<br>
-> MongoDB – NoSQL database<br>
-> Mongoose – Database modeling<br>
-> Axios – API communication<br>
-> JWT (JSON Web Token) – Authentication & security<br>
-> Socket.io – Real-time updates<br>
-> HTML5, CSS3, Bootstrap – Responsive UI design<br>
-> npm – Package management<br>

⚙️ System Workflow<br>

-> User registers / logs in securely<br>
-> Access dashboard with inventory overview<br>
-> Add / update / delete inventory items<br>
-> Track stock inflow and outflow<br>
-> System updates stock status automatically<br>
-> Real-time updates using Socket.io<br>
-> View stock alerts and trends<br>
